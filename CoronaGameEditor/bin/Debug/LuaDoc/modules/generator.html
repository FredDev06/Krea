Tables

Fields
Members of the Generator Instance

Fields:

  • params: The table params used to create the generator instance
  • name: The name of the generator
  • objectAttached: The fastener of the generator if existing
  • objectToDuplicate: The Object instance to duplicate
  • iteration: The number of object to generate (-1 = Infinite)
  • delay: The delay in milliseconds between two object generation
  • copyPhysicBody: Indicates whether the generator should also copy the physics body of the object to duplicate
  • map: The instance of a Tiles map if existing in the scene
  • isGenerating: Indicates whether the generator is generating objects or not
  • tabObjects: A table containing all object instances generated by the generator
