Fields
Members of the Radar Instance

Fields:

  • params: The table params used to create the radar instance
  • tabObjects: The multidimensional table containing the different categories of objects to display on the radar
  • surface: The display object used as display radar surface
  • sceneWidth: The scene width to display on the radar
  • sceneHeight: The scene height to display on the radar
  • tabEchoColors: The multidimensional table containing the different echo category colors to display on the radar
  • echoAlpha: The alpha of the radar echos
  • refreshInterval: The refresh interval of the radar
  • isEnabled: Indicates whether the radar is enabled or not
