Fields
Members of the Object Instance

Fields:

  • name: The name of the object instance
  • object: The Corona display object of the object instance
  • displayGroupParent: The display group where is stored the display object
  • params: The table params used to create the object instance
  • type: The display object type of the object instance
  • xOrigin: The x origin location of the object instance
  • yOrigin: The y origin location of the object instance
  • hasBody: Boolean that indicates whether the display object has a physics body or not
