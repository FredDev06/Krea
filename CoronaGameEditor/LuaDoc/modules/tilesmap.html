Module tilesmap

Tilesmap is a module providing the possibility to add into a project the concept of TilesMap.
This one has a real interest if you want a big world in your project without using too much the memory.
Changing the content of this file is at your own risk. As the software Krea uses this file to auto-generate code, it can create errors during generation if you change his content.

Tables

Fields
Members of the Tiles Map Instance

Fields:

  • displayGroup: The Corona display group where each tile is stored
  • name: The tiles map name
  • isInfinite: Boolean which indicates whether the tiles map is infinite or not
  • infos: A table containing informations about the tilles map
  • isPhysicsEnabled: Boolean which indicates whether the tiles map interacts with physics
  • nbTilesX: The tiles count on X axis
  • nbTilesY: The tiles count on Y axis
  • tileWidth: The width of each tile
  • tileHeight: The height of each tile
  • textureSet: The sprite set containing all textures (First tile layer)
  • imageObjectSet: The sprite set containing all objects textures (Second tile layer)
  • texturesInitFilePath: The path of the textures initialisation file
  • objectsInitFilePath: The path of the objects initialisation file
  • tilesCollisionInitFilePath: The path of the collisions initialisation file
  • xScroll: The current value of the tiles map scroll on the X axis
  • yScroll: The current value of the tiles map scroll on the Y axis
  • isPaused: Indicates if the tiles map is in a paused state
  • tabTiles: The table containing each tile (Corona sprite object)
  • tabObjectsInInteraction: The table containing all external display objects handled by the tiles map
  • xOrigin: The X origin of the tiles map
  • yOrigin: The Y origin of the tiles map
  • xTilesInView: The number of tile that can be viewed from the screen on X axis
  • yTilesInView: The number of tile that can be viewed from the screen on Y axis
